# onNextStart
#### `onNextStart` will pass both the __current__ and __next__ `item-object`:  
```
(currentItem, nextItem) => {/*...*/}
```
- The `item-object` has the shape of:
```
{
    item: {/* your item */},
    index: // the item's index number 
}
```
