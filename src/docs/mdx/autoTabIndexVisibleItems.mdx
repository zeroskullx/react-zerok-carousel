# autoTabIndexVisibleItems
Automatically inject `tabIndex:0` to visible items and `tabIndex:-1` to none visible items.  

**This prop defaults to `true`.**

In some cases you might want users to directly focus nested elements inside the Item, 
so you can grab that `tabIndex` and pass it to whatever nested element you wish.

In this example we pass the injected `tabIndex` to the `<a/>` element:

<Playground >
    <input placeholder="start focus from here..." />
    <Carousel itemsToShow={3} autoTabIndexVisibleItems={true}>
        <ItemWithLink text="1" />
        <ItemWithLink text="2" />
        <ItemWithLink text="3" />
        <ItemWithLink text="4" />
        <ItemWithLink text="5" />
        <ItemWithLink text="6" />
    </Carousel>
</Playground>
