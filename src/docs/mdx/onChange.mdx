# onChange
#### `onChange` will pass the new __current__ `item-object` and __current__ `page-index`:  
```
(currentItem, pageIndex) => {/*...*/}
```
- The `item-object` has the shape of:
```
{
    item: {/* your item */},
    index: // the item's index number 
}
```
